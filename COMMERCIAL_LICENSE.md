# Commercial License

HUMAN LEGAL REVIEW REQUIRED

This file documents the intended commercial boundary for the first paid self-hosted release.

## License strategy note

This repository intentionally treats license choice as a strategy decision first. Options considered include:

- public demo repository plus separate commercial distribution
- Business Source License 1.1
- custom commercial license or EULA

For this first paid release draft, the repository includes this commercial-license placeholder so the boundary is explicit even before final legal review.

## Draft commercial intent

- the buyer receives a perpetual right to install and run the purchased release for self-hosted use
- the runtime remains buyer-owned and buyer-operated
- secrets, data, and external API costs remain under the buyer's control

## Not granted by this draft

- redistribution rights
- resale rights
- sublicensing rights
- hosted-service resale rights
- any promise of an open-source license

## Draft update posture

- one-time purchase
- limited updates
- no subscription implied by the repository alone

## Needs confirmation before sale

- number of permitted installs
- whether one buyer may run multiple internal copies
- whether source redistribution is allowed
- support window duration
- update entitlement duration
- governing law, warranty, and limitation-of-liability language
